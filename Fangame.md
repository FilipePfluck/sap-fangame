This project is a Fan game. It has no association with Team Wood Games. 

A group of players created fan made packs with fan made pets and foods. We want to implement it as a fan game.

Players will only be able to use free pets and foods and our fan made ones.

We will develop the game with NextJS for both frontend and backend. We will use Prisma to connect to the database. The database will be hosted on Supabase and the app will be deployed in vercel. Tests will be written with Vite.
