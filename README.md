This is a Super Auto Pets fangame. It has no relation to Team Wood Games.

## Getting Started

Clone the repository.

Install the dependencies:

```bash
npm i
```

Add the .env credentials (ask them on discord)

Run the development server:

```bash
npm run dev
```

## How to contribute

Ask me to add you to the project in the github.

Create a branch and commit your changes. Then open a pull request and I will review it.
